Install Nanco Paints

To install Nanco Paints as an app on your device, follow these steps:

1

On iOS Safari: Tap the Share button in the bottom menu, scroll down, and tap Add to Home Screen.

2

On Android Chrome: Tap the menu and select Install App or Add to Home Screen.

3

On Desktop (Chrome/Edge): Click the Install icon or similar icon in the address bar at the top of the window.
